## 프로젝트 소개
#### 기간 : 2024.08.09 ~ 2024.10.11 (약 7주)
#### 주요 기능
1. **프리랜서 맞춤 프로젝트 추천**
2. **AI 기반 프로젝트 자동 등록**
3. **프로젝트 이상거래 탐지**
4. **NFT를 이용한 계약 및 경력 증명**
5. **쿠버네티스를 이용한 재난 방지 및 고가용성**
6. **금융 및 유저 서비스를 분리한 MSA 아키텍처**
